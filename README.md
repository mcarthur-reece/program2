## Setup Instructions
1. Clone the repository
2. Run `bundle install`
3. Run `rails db:create db:migrate db:seed`
4. Run `rails server`
5. Navigate to `http://localhost:3000`

## Database Schema
- Volunteers table: user accounts for volunteers
- Admins table: admin account (pre-seeded)
